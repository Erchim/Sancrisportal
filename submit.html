Submit an event / tip

Fast workflow: you collect, the site publishes. (For now: manual copy into JSON.)

  1. Open the submission form: data/site.json → submit_form_url
  2. When you receive a submission, add it to data/events.json
  3. Commit to GitHub — site updates automatically
