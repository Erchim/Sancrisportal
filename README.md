# San Cristóbal Guide — Package A

## Edit content
- Events: data/events.json
- Life: data/articles.json + posts/*.md
- Nature: data/places.json + posts/*.md
- FAQ: data/faq.json
- Hotels / Restaurants / Music: data/*.json

## Featured / Sponsored
Use `featured: true` and optional `priority` in JSON.
