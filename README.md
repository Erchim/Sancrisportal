# SanCris Portal (v5)

Static website for GitHub Pages (no backend).

## Change branding / links
Edit: `data/site.json`

## Add content
- Life articles: `data/articles.json` + `posts/<slug>.md`
- Nature places: `data/places.json` + `posts/<slug>.md`
- Events: `data/events.json`
- Hotels/Restaurants/Music: `data/*.json`

## Images
Put images in `assets/images/` and reference by path, e.g.:
- `assets/images/my-photo.jpg`

See: `CONTENT_GUIDE.md`


## Supabase (community features)
See `SUPABASE_SETUP.md` to connect Supabase from zero.
